# useReducerWithLocalStorage

useReducerWithLocalStorage

## Install

```bash
yarn add use-reducer-with-local-storage
```

## Usage

```js
const [state, dispatch] = useReducerWithLocalStorage({
  initializerArg: emptyState,
  key: 'REACT_APP_STATE',
  reducer
});
```
